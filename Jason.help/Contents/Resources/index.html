Quick Manual

Application created on fork of old opensource Jason editor by Olivier Labs (2010)

However application was significantly improved according to current needs.

Among others:

Undo/redo
Copy/paste
Drag and drop
Autosaving and versioning
Search
Cross-reference search to reveal value by keys
etc

Please mind that this editor didn't edit JSON text explicitly.
It decompose JSON into base model components as TreeNodes, and then compose a new JSON text document while saving. It retain order of elements in dictionaries and arrays, however it is forbidden to have several values in one dictionary with same key, so nevertheless it is possible to have tree structure presented that way on UI, values will be overriden during saving.

Quick keys:

tad edit key for selected row
enter edit value for selected row, if available
cmd+I insert new row below selected row
alt + select row reveal value: search all opened files for key that contains value of selected row
cmd + E reveal value (see above)
cmd + ]/[ jump between different revealed values
cmd + F open Find bar
